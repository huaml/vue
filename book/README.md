# Book

> Vue.js 简单小案例-Book App

## 安装与运行
### 克隆项目
```
git clone https://github.com/huaml/vue.git
```
### 进入项目文件
```
cd vue/book
```
### 安装依赖
```
npm install
```
### 启动项目
```
npm run dev
```
### 打包部署
```
npm run build
```
## 数据来源
```
data.json模拟数据
```
## 技术
```
html, sass, vue.js, webpack
```
