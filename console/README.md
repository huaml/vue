## 使用

### 克隆项目
```javascript
git clone https://github.com/huaml/Bootstrap
```

### 安装依赖

```javascript
npm i
```

### 运行

```javascript
gulp
```
自动打开浏览器访问：`http://localhost:3000/`。 Browsersync 带热重载功能。
